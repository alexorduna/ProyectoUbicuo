
# ProyectoUbicuo — Sistema CAA con Reconocimiento de Gestos

Aplicación móvil nativa en Swift que funciona como sistema de Comunicación Aumentativa y Alternativa (CAA). Utiliza la cámara del dispositivo y modelos de visión por computadora on-device para rastrear e interpretar gestos manuales en tiempo real, traduciéndolos en frases mediante síntesis de voz (Text-to-Speech).

## Problemática
Las personas con mutismo o discapacidades severas del habla enfrentan barreras significativas en entornos laborales dinámicos. Las herramientas actuales exigen desviar la mirada, ocupar ambas manos y dedicar tiempo a formular mensajes, interrumpiendo el flujo de trabajo.

## Solución
Un MVP que permite comunicación fluida mediante gestos manuales, sin interacción táctil con la pantalla, aprovechando únicamente el dispositivo móvil de uso diario.

## Tecnologías
- Swift (iOS nativo)
- Vision Framework / MediaPipe (detección de mano y landmarks)
- AVFoundation (captura de video)
- AVSpeechSynthesizer (Text-to-Speech)
